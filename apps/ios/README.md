# iOS App

SwiftUI app for Sami's Guide — personal place saving, maps, and Instagram import.

## Stack (planned)
- **SwiftUI** + MVVM
- **MapKit** for maps
- **Vision** for on-device OCR (screenshot imports)
- **Share Extension** for URL + image intake
